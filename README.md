# 🚀 Multi-Agent Content Storyboard SaaS

An AI-driven automation pipeline that transforms market topics into comprehensive multimedia storyboards. This system utilizes a multi-agent architecture to analyze sentiment, research news, and generate audio-visual production plans.

**Live Demo:** [multi-agent-saas.vercel.app](https://multi-agent-saas-oioruanzz-saurabhkaran11s-projects.vercel.app/)

<img width="1375" height="685" alt="Dashboard Preview" src="https://github.com/user-attachments/assets/f4215f4e-cc97-4530-b8b0-6af2c2fbd190" />

## 🌟 Features

- **Multi-Agent Orchestration**: Powered by `TokenRouter` and `OpenAI` models (Qwen, GLM) to handle complex reasoning tasks.
- **Sentiment Engine**: Analyzes market topics using weighted keyword scoring and Optional HuggingFace RoBERTa models.
- **Automated Storyboarding**: Generates video titles, scripts, and visual prompts automatically based on market bias.
- **Multimedia Integration**: Support for ElevenLabs (Audio), Replicate/Runway (Video), and News Research APIs.
- **Persistence Layer**: Full integration with Supabase for real-time ledger tracking of generated storyboards.
- **Deployment Ready**: Configured for Vercel, Docker, and Zeabur.

## 🛠 Tech Stack

- **Backend**: Python 3.11, FastAPI, Uvicorn
- **AI/LLM**: TokenRouter (Qwen-3.5, GLM-5.1), OpenAI SDK
- **Database**: Supabase (PostgreSQL)
- **Deployment**: Vercel Serverless, Docker
- **Data Validation**: Pydantic v2 (Settings & Models)

## 📋 Prerequisites

- Python 3.11+
- Supabase Project (with `content_storyboards` table)
- API Keys for: TokenRouter, Supabase, Market Data, News Research.

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Saurabhkaran11/multi-agent-saas.git
   cd multi-agent-saas
   ```

2. **Set up Environment**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```

3. **Configure Secrets**:
   Create a `.env` file with your `SUPABASE_URL`, `SUPABASE_KEY`, and `TOKENROUTER_API_KEY`.

4. **Run Locally**:
   ```bash
   uvicorn main:app --reload
   ```

## 🏗 Database Schema

Run the following SQL in your Supabase SQL Editor:

```sql
CREATE TABLE IF NOT EXISTS content_storyboards (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    topic_headline TEXT NOT NULL,
    sentiment_score FLOAT NOT NULL,
    market_bias VARCHAR(50) NOT NULL,
    video_title TEXT NOT NULL,
    audio_script TEXT NOT NULL,
    visual_prompts TEXT NOT NULL,
    media_audio_url TEXT DEFAULT NULL,
    media_video_url TEXT DEFAULT NULL,
    status VARCHAR(50) DEFAULT 'Pending Video Production'
);
```
